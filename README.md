# 🔐 Data Governance & Security Dashboard | Power BI Project

## 📌 Overview
The **Data Governance & Security Dashboard** is an interactive Business Intelligence project developed using **Microsoft Power BI** during my Data Analytics Internship at **Unified Mentor**.

This project transforms complex Environmental, Social, and Governance (ESG) datasets into meaningful visual insights that help users monitor governance performance, analyze trends, and compare countries through interactive dashboards.

The project demonstrates real-world data analytics workflows including data preparation, modeling, visualization, and analytical storytelling.

---

## 🚀 Key Features
- Interactive governance performance dashboard
- Country-wise comparison analysis
- Time-series trend visualization
- Dynamic KPI metrics using DAX
- Interactive slicers and filters
- Clean and professional dashboard design

---

## 🛠️ Tech Stack
- Microsoft Power BI
- Power Query Editor
- DAX (Data Analysis Expressions)
- Excel & CSV Data Sources
- Star Schema Data Modeling

---

## 📊 Dashboard Insights
The dashboard provides:

- Governance Performance Overview
- Country-wise Governance Analysis
- Indicator Category Analysis
- Trend Analysis Over Time
- Comparative Performance Insights

Users can explore governance indicators across countries and years to support data-driven decision-making.

---

## 📁 Dataset Information
The project integrates multiple ESG datasets:

- **ESGData.csv** — Governance indicator values
- **ESGCountry.csv** — Country metadata
- **ESGSeries.csv** — Indicator descriptions
- **ESGSeries-Time.csv** — Time mapping
- **ESGCountry-Series.csv** — Country-indicator relationships
- **ESGFootNote.csv** — Supporting metadata

## 🧩 Data Modeling
A **Star Schema** model was implemented:

- Fact Table → Governance Indicators
- Dimension Tables → Country, Indicator, Time

This structure improves performance and analytical clarity.
